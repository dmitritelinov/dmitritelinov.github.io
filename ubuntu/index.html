Ubuntu

Ubuntu is an open source operating system software for computers. It is one of the distribution systems of Linux, and is based on the Debian architecture.


Ubuntu is published by Canonical Ltd, who offer commercial support. It is based on free software and named after the Southern African philosophy of ubuntu (literally, 'human-ness’).
