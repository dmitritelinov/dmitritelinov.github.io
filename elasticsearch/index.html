Elasticsearch

Elasticsearch is a distributed, multitenant-capable full-text search engine with an HTTP web interface and schema-free JSON documents.


Elasticsearch is developed in Java and is released as open source under the terms of the Apache License. Official clients are available in Java, .NET (C#), Python, Groovy and many other languages. Elasticsearch is the most popular enterprise search engine.
