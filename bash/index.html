bash

bash is a free software Unix shell written for the GNU Project. Its name is an acronym which stands for Bourne-again shell.


The name is a pun on the name of the Bourne shell (sh), an early and important Unix shell written by Stephen Bourne and distributed with Version 7 Unix circa 1978 and the phrase born again. bash was created in 1987 by Brian Fox. In 1990 Chet Ramey became the primary maintainer.
