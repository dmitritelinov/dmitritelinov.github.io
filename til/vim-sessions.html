Vim: sessions

Sessions are project-specific states for Vim. The session contains:

  • Blank windows
  • Buffers
  • Current directory
  • Folds
  • Help
  • Options
  • Mappings
  • Window sizes
  • Window layouts
  • Tabs

To set/get information use:
:wviminfo
:rviminfo

:mksession! ~/.vim/sessions/book.vim - save session
:source ~/.vim/sessions/job.vim - load/reload a session
vim -S ~/.vim/sessions/book.vim - load a session on startup
vim --clean - start clean vim environment
