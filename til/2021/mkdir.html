mkdir

Allows users to create new directories. With mkdir, you can also set permissions, create multiple folders at once, and much more.

Make multiple directories. Add -v option for verbose output: 

mkdir {test1,test2,test3}
Building a structure with multiple subdirectories requires adding the -p option: 
mkdir -p parent/{child1,child2}
The mkdir command by default gives 755 permissions - or the umask setting 022.
To create a folder with 700 permissions: 
mkdir -m700 directory
