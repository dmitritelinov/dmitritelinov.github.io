VSCode setup to support Kubernetes manifest files

  • Go to Extenstions and install YAML extenstion from RedHat - YAML Language Support by Red Hat
  • Press Ctrl+P search for settings.json, open the file for editing and add the following lines under yaml.schemas: 
    "yaml.schemas": {
      "kubernetes": "*.yaml"
    }
    If you cannot find the file - go to plugin settings and scroll to the bottom to Yaml: schemas and click on Edit in settings.json
  • Restart editor
