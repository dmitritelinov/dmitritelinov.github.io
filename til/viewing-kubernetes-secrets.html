Viewing Kubernetes Secrets

To view encoded secrets use: 

$ kubectl get secret app-secret -o yaml
apiVersion: v1
kind: Secret
metadata:
  name: app-secret
data:
  APP_MODE: cHJvZA==
  APP_PASS: cGFzc3dvcmQ=
To decode the values use: 
$ echo -n "cHJvZA==" | base64 -d
prod
$ echo -n "cGFzc3dvcmQ=" | base64 -d
password
