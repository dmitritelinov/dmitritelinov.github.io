lsblk

Show all block devices which is giving an overview of a disks as well as partitions in Linux 

$ sudo lsblk
NAME        MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT
sda           8:0    0  10.9T  0 disk 
└─sda1        8:1    0  10.9T  0 part /mnt/ntfs
nvme0n1     259:0    0 465.8G  0 disk 
├─nvme0n1p1 259:1    0 401.8G  0 part /
├─nvme0n1p2 259:2    0     1K  0 part 
└─nvme0n1p5 259:3    0  31.9G  0 part [SWAP]
