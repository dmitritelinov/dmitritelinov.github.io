influxdb grep

Drop multiple measurements from InfluxDB database using a pattern

influx --execute 'show measurements' --database={{ db_name }} | grep {{ m_pattern }} |xargs -I{} \
influx --database={{ db_name }} --execute 'drop measurement "{}"'
{{ db_name }} - is a InfluxDB database name, ex: idata
{{ m_pattern }} - a measurement pattern to grep for, ex: "^app-"

It will show all measurements from {{ db_name }} InfluxDB database, grep them by {{ m_pattern }} and finally drop the matched measurements
