Environment variables in bash

Define: 

export DB_USER=demo
Print the value: 
printenv DB_USER
echo $DB_USER
env | grep DB_USER
printenv |grep DB_USER
Delete/unset: 
unset DB_USER
Define at user level:
add the export statement from Define to the ~/.bashrc Define at global level:
add the export statement from Define to the /etc/environment
