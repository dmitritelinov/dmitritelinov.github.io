Mount volumes and directories in Docker - new and old way

Old way
Volume: 

docker run \
-v data_volume:/var/lib/mysql \
mysql
Bind: 
docker run \
-v /data/mysql:/var/lib/mysql \
mysql
New way
Volume: 
docker run \
--mount source=data_volume,target=/var/lib/mysql \
mysql
Bind: 
docker run \
--mount type=bind,source=/data.mysql,target=/var/lib/mysql \
mysql
