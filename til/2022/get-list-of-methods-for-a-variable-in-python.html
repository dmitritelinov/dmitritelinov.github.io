Get list of methods for a variable in Python

For this - you can use the dir() function.

Example: 

num = 89
str = "Dmitri"
bl = False
fl = 3.14

print("int methods:")
print(type(num))
print(dir(num))

print("str methods:")
print(type(str))
print(dir(str))

print("bool methods:")
print(type(bl))
print(dir(bl))

print("float methods:")
print(type(fl))
print(dir(fl))
