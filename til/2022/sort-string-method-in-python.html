sort string method in Python

This function will sort in place.
Example: 

my_list = [15, 6, 7, 8, 35, 12, 14, 4, 10]

print(f"Ints: {my_list}")

print("Sorting...")
my_list.sort()
print(f"Sorted Ints: {my_list}")
Output: 
Ints: [15, 6, 7, 8, 35, 12, 14, 4, 10]
Sorting...
Sorted Ints: [4, 6, 7, 8, 10, 12, 14, 15, 35]
