Vim: Text manipulation

> - shift text left
< - shift text right
>> - shift text left by shiftwidth
<< - shift text right by shiftwidth
== - auto indent line
~ - change case
7~ - toggles case for 7 characters, or hold down ~ to repeat
gul - similar to ~
gUw - changes case for current word
gU3w - changes case for 3 words from cursor
gUU - changes entire line to UPPER
guu - changes entire line to lower
Shift+u - on highlighted text, make text uppercase
Shift+j - join current and next lines
